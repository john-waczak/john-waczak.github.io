Linear Transformations as a Vector Space

To begin, recall the definition of a linear transformation between two vector spaces \(V\) and \(W\), i.e. a mapping \(\mathscr{L}:V\to W\) which satisfies \[\begin{aligned} &\mathscr{L}\left(\vec{u}+\vec{v}\right) = \mathscr{L}\left( \vec{u} \right)+\mathscr{L}\left( \vec{v} \right) \\ &\mathscr{L}\left( \alpha \vec{u} \right) = \alpha\mathscr{L}(\vec{u})\end{aligned}\] for any \(\vec{u},\vec{v} \in V\) and \(\alpha \in \mathbb{R}\). We note that operations on the left hand side occur in \(V\) while operations on the right hand side occur in \(W\).

Having defined individual linear transformations, we now consider the set of all linear transformations between \(V\) and \(W\) which we denote \(\mathscr{L}(V,W)\). For any \(\phi, \;\psi\in \mathscr{L}(V,W)\), we define \[\begin{aligned} &(\phi+\psi)\vec{v} = \phi(\vec{v}) + \psi(\vec{v})\\ (\alpha \phi)(\vec{v}) = \alpha\phi(\vec{v})\end{aligned}\]
